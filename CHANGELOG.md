## [0.1.0] - 2026-02-10

### 新增

- 初始版本发布
- 基于 ECDICT 的本地 SQLite 词典查询
- 支持 Free Dictionary API 在线查询作为后备
- 模糊搜索和拼写建议
- 中文释义、英文定义、例句展示
- 词形变换信息显示
- 交互式词库下载和本地导入功能
- 彩色格式化输出
